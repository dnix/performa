# Performa: Real Estate Financial Modeling Library

Performa is a modern Python library for real estate financial and data modeling. It provides a set of standardized models based on Pydantic structures, aiming to create a living open standard for the real estate industry.

## Features

- Standardized models for various aspects of real estate finance
- Built on modern Python conventions and Pydantic for robust data validation
- Flexible and extensible architecture
- Comprehensive set of tools for real estate financial analysis

## Installation

Currently, Performa can be installed directly from the source code. To install, follow these steps:

1. Clone the repository:

   ``` python
   git clone https://github.com/your-repo/performa.git
   cd performa
   ```

2. Install the package in editable mode:

   ``` python
   pip install -e .
   ```

This will install Performa in editable mode, allowing you to make changes to the source code and immediately see the effects without needing to reinstall.

In the future, Performa will be available on PyPI, enabling installation via pip:

``` python
pip install performa
```
