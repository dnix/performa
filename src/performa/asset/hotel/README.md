# Performa Hotel Asset Models - Hospitality Property Analysis

This module will provide comprehensive modeling capabilities for hospitality
properties, including hotels, resorts, and extended-stay facilities.

## Planned Features

### Core Models (Future Implementation)
- **HotelProperty**: Main property container with room mix and hospitality operations
- **HotelRoom**: Individual room specifications with hospitality-specific attributes
- **HotelAnalysisScenario**: Analysis orchestration for hospitality cash flows

### Hospitality-Specific Components
- Room revenue modeling with ADR (Average Daily Rate) and occupancy
- Food & beverage revenue integration
- Seasonal demand patterns and market cycles
- Management fee structures (base + incentive)
- Hotel operating expense categories (rooms, F&B, administrative)

### Key Features (Planned)
- RevPAR (Revenue Per Available Room) optimization
- Seasonal and cyclical demand modeling
- Hotel management agreement integration
- Capital expenditure planning for FF&E (Furniture, Fixtures & Equipment)
- Market positioning and competitive analysis

## Integration

Will integrate with existing Performa modules:
- **Core primitives** for timeline and cash flow modeling
- **Debt module** for hotel financing structures
- **Deal module** for hotel acquisitions and partnerships
- **Valuation module** for hospitality-specific metrics
